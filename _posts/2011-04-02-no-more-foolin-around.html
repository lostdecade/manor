We've reverted our April Fool's version of Onslaught! Arena back to the game we all know and love. Chrome Web Store users should see the game update in the next day or so.

We enjoyed the Super Meat Boy mode so much that we've left it in game, accessible via a keyboard code. Simply type MEAT anytime during the game to play as Super Meat Boy!

Additionally, the game still saves every wave now. We realize the game is extremely difficult so now attempting the Dragon or Beholder over and over until you get it right won't be quite as punishing! :)

We gave away Onslaught! Arena on Mac App store for free yesterday as well and 646 lucky users snagged a copy! If you liked it please leave a review!

Thanks everyone!
